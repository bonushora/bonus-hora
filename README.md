# Bonus-Hora

Bonus-Hora é uma moeda social baseada em tempo destinada a facilitar trocas de serviços em comunidades.

## Problema
Muitas pessoas possuem habilidades mas não têm acesso a dinheiro para trocar serviços.

## Solução
Criar um sistema digital de bônus-hora onde:

1 hora de serviço = 1 token

## Funcionalidades

- Registro de serviços
- Carteira digital
- Sistema de reputação
- Mediação de conflitos
- Governança comunitária (DAO)

## Tecnologias

- Blockchain
- Smart Contracts
- QR Code
- Aplicativo mobile

## Objetivo

Criar hubs de trocas comunitários que fortaleçam economias locais.
