# Project Type

Bonus-Hora is a Web3 social economy project.

Category: Social Currency / Time-based Economy

The system proposes a decentralized infrastructure where:

1 hour of service = 1 Bonus-Hora token

The platform may include:

• blockchain-based transaction records
• community governance mechanisms
• decentralized identity and reputation systems
• mobile applications for service exchange

The goal is to build community-driven economic networks based on time, trust and collaboration.
