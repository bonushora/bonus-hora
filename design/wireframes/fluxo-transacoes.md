# Fluxo de Transações

O fluxo de transações do sistema BonusHora segue as etapas abaixo:

1. Cliente solicita um serviço.
2. Prestador aceita a solicitação.
3. Serviço é executado.
4. Cliente confirma a execução.
5. O sistema transfere BonusHora do cliente para o prestador.
6. Ambos podem avaliar a experiência.

Status possíveis da transação:

- Pendente
- Confirmado
- Cancelado
