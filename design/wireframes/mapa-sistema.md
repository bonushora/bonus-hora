
# Mapa do Sistema BonusHora

O sistema BonusHora é uma plataforma digital para trocas comunitárias baseadas em tempo de trabalho.

Estrutura geral do sistema:

Tela inicial  
↓  
Login ou Cadastro  
↓  
Identificação do tipo de usuário  

Tipos de usuário:

- Superadministrador
- Administrador
- Prestador/Cliente

Cada usuário é redirecionado automaticamente para seu painel.

Painéis disponíveis:

- Painel Superadmin
- Painel Administrador
- Carteira do Usuário

O sistema registra todas as transações de serviços prestados entre usuários.
